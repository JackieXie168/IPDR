Custom error messages

Collector error messages

Following custom error messages are sent by the collector:

256
An Outputter error has occured
257
Generic/unknown session error has occured

Exporter error messages

256
BrokenTemplate - Sent after a MODIFY TEMPLATE message is received when the negotiated template arrived did not decoded well by the exporter
257
CollectorNotConfigured - Sent after a FLOW START, MODIFY TEMPLATE, FINAL TEMPLATE DATA ACK or DATA ACKNOWLEDGE is received from a non configured (for the session) collector. This information is obtained from the returned value of: ExporterProtocol_CollectorSession
258
InvalidConnectionState - Sent when connection state is not valid for the received message
