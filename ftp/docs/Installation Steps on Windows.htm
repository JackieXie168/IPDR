Windows Build Instruction for VisualC++ Compilation of the IPDR Libraries

Dependent Libraries

The reference libraries were built and tested using version 7.9.3 of libcurl for URL implementation and version 1.95.2 of Expat for the XML parser.  It is also dependent on the base IPDR library.

Build Instructions

  1. After downloading curl and expat, you'll still need to build libcurl.  Expat packages the compiled lib, so you won't need to build it.  Note the current tested versions are older ones.  For libcurl, you will need to pick the "older releases" to find a copy of the version used.
  2. Curl provides a .dsw (workspace) file at lib/curllib.dsw.  Open this with VisualC++ and do the build.
  3. To work with the default .dsw packaged with ipdrftp, move the following files:
    • copy curl's lib/Debug/libcurl.lib built in step 2 to ipdrftp's libs directory
    • copy curl's include/curl directory and its contents to ipd's include area, creating an include/curl subdirectory.
    • copy expat's libs/expat.lib to ipdrftp's lib directory.
    • copy the header file from expat's source/lib/expat.h to the directry include/expat in the ipdrftp project.
    • copy the header file directories from the base IPDR project to subdirectories to the ipdrftp include area.
  4. Open the IPDRFTP .dsw (workspace) file at lib/ipdrftp.dsw and build.
  5. Open the IPDRFTP test .dsw file at testWin32.dsw and build both the various consumer and producer projects
  6. Create a bin directory in IPDRFTP.  Copy the ipdrftp.dll file to this directory, finally copy the libcurl.dll and expat.dll to this directory.
  7. Modify your classpath add the ipdrftp/bin directory.  This will allow you to simply execute the various consumer and producer test.
  8. Test the program by doing the following:
    • Get a DOS prompt.
    • cd <ipdrinstalldir>/test
